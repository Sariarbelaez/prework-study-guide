Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch name: creates a new branch and switches to it.
  • cd: takes you to a repository.
  • Clear: clears all progress.
  • git clone: retrieves an entire repository from a hosted location via URL.
  • git config: This command sets the author name and email adress
  • git init: used to start a new repository
  • git commmit -m: command records or snapshots the file permanently in the version history.
  • git commit -A: command commits any files you've added with the git add command and also commits any files you've changed since then.
  • git rm: command deletes file from working directory and stages the deletion.

JavaScript

  • JavaScript is a very powerful programming language that can add interactivity to a website.
  • It can be used to creare games, animated 2D and 3D graphics.
  • Variable: is a symply named container that allows us to store date in our code.
  • Var: allows us to declare a variable.

  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is executed from top to bottom. However using conditional statements such as IF statements allows us to interrupt control flow.
  • Array: A single variable that is used to hold a gorup of data.
  • Var: A way to create an Array.
  • For: uses the predictable pattern of indices to perform a task.
